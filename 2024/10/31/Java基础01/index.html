数据类型

  • 强类型语言 Java、c++

    • 要求变量的使用严格规定，所有变量必须先定义再使用
    • 安全性高，速度慢

  • 弱类型语言 VB script、Java script

  • Java的数据类型分为两大类

    • 基本类型（primitive type)
      • 数据类型
        • 整数类型
          • byte，1个字节范围：-128~127
          • short，2个字节范围：-32768~32767
          • int，4个字节范围：-2147483648~2147483647（够了）
          • long，8个字节范围：-9223372036854775808~9223372036854775807
        • 浮点类型
          • float 占四个字节
          • double 占八个字节(常用）)
        • 字符类型-char占两个字节
      • boolean类型 占一个位
        • true
        • false
    • 引用类型（reference type)
      • 类
      • 接口
      • 数组

什么是字节

  • 位bit：是计算机内部数据储存的最小单位，11001100是一个八位二进制数

  • 字节Byte：是计算机中数据库处理的基本单位，习惯上用大写B来表示

  • 1B=8bit

  • 字符：是计算机中使用的字母、数字和符号

  • 1KB=1024B

  • 1M=1024KB

  • 1G=1024M

数据类型扩展

  • 整数扩展

    • 二进制
    • 十进制
    • 八进制0
    • 十六进制0x 0-9 A-F 16

  • 浮点数拓展

    • Float 有限 离散 舍入误差
    • 最好不要用浮点比较
    • 要比较用BigDecimal

  • 字符类拓展

    • 强制转换 unicode编码 65535 U0000-UFFFF

    • 所有字符本质还是数字

    • 转义字符

      转义字符 意义 ASCII码值（十进制）
      \a 响铃(BEL) 007
      \b 退格(BS) ，将当前位置移到前一列 008
      \f 换页(FF)，将当前位置移到下页开头 012
      \n 换行(LF) ，将当前位置移到下一行开头 010
      \r 回车(CR) ，将当前位置移到本行开头 013
      \t 水平制表(HT) （跳到下一个TAB位置） 009
      \v 垂直制表(VT) 011
      \ 代表一个反斜线字符”" 092
      ' 代表一个单引号（撇号）字符 039
      " 代表一个双引号字符 034
      ? 代表一个问号 063
      \0 空字符(NUL) 000
      \ddd 1到3位八进制数所代表的任意字符 三位八进制
      \xhh 十六进制所代表的任意字符 十六进制

    • 对象 从内存分析

  • 布尔值扩展

类型转换

  • Java是强类型语言，运算时要类型转换

  • Byte,short–>int–>long–>float–>double

    低——————————————>高 字节大小

  • 强制类型转换高到低

  • 自动类型转换低到高

  • 注意点

    1. 不能转换布尔值
    2. 不能转换对象类型为不相干类型
    3. 转换可能存在内存溢出和精度问题
    4. 操作较大数时，注意溢出问题

  • 计算

变量

  • 可以变化的量
  • Java是强类型语言，每个变量必须声明类型
  • Java变量是程序中最基本的储存单元，其要素包括变量名，变量类型，==作用域==

变量作用域

  • 类变量

  • 实例变量

  • 局部变量

常量

  • 常量Constant：初始化initialize后不能再变化的值

  • 常量可以理解成一种特殊的变量，它的值被设定后，在程序运行过程中不允许改变

  • 常量名一般使用大写字符

变量的命名规范

  • 所有的变量、方法、类名：见名知意

  • 类成员变量：首字母小写和驼峰原则

    monthSalary 除了第一个单词以外，后面的单词首字母大写

  • 局部变量：首字母小写和驼峰原则

  • 常量：大写字母的下划线MAX_VALUE

  • 类名：首字母大写和驼峰原则Man,GoodMan

  • 方法名：首字母小写和驼峰原则：run(),runRun()

运算符

  • Java语言支持如下运算符：优先级()
    • 赋值运算符：+,-*,/,%(模/取余),++,–
    • 赋值运算符=
    • 关系运算符:>,<,>=,<=,==（等于）,!=（不等于）instanceof
    • 逻辑运算符：&&（与）,||（或）,!（非）
    • 位运算符:&,|,^,~,>>,<<,>>>
    • 条件运算符?:
    • 扩展赋值运算符:+=,-=,*=,/=

出现小数时进行类型转换，否则会丢失精度

有double输出double,无double输出lonf,无long输出int，无int也输出Int

包机制

  • 为了更好地组织类，Java提供了包机制，用于区分类名的命名空间（本质是文件夹）

  • 包语句的语法格式为

  • 一般利用公司域名倒置作为包名；www.baidu.com com.baidu.www

  • 为了能够使用MV一个包的成员，我们要在Java程序中名犬导入该包。使用”import”语句可以完成此功能

JavaDoc

  • JavaDoc命令是用来生成自己的API文档
  • 参数信息
    • @author 作者名
    • @version 版本号
    • @since 知名需要最早使用的JDK版本
    • @param 参数名
    • @return 返回值情况
    • @throws 异常抛出情况
